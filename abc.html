Course Introduction

This NLP course provides a solid foundation in language models, text processing, sentiment analysis, and transformer architectures, emphasizing both conceptual understanding and practical implementation.

Course Focus

Students explore language understanding, model design, and the ethical use of NLP systems. The course combines project-based learning and hands-on experimentation to strengthen understanding.

Course Units

Unit 1: Introduction to NLP

  • Overview of NLP applications
  • Text representation and preprocessing
  • Tokenization, stemming, and lemmatization

Unit 2: Language Modeling

  • n-gram models and smoothing
  • Word embeddings (Word2Vec, GloVe)
  • Transformers and contextual embeddings

Unit 3: Text Classification and Sentiment Analysis

  • Feature extraction with TF-IDF
  • Sentiment analysis techniques
  • Case studies using IMDB dataset

Unit 4: Sequence Models

  • RNNs, LSTMs, and GRUs
  • Language generation
  • Named Entity Recognition

Unit 5: Applications and Ethics

  • Machine translation
  • Summarization techniques
  • Bias and fairness in NLP systems
